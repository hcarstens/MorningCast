# Guide My Day

Build a prototype for a  new game

The new game has three main divinators:

horoscope, tarot, iching

And returns a single sign: love, fortune, fame, adventure

Idea: Every day you receive three readings: horoscope, tarot and iching

Which combine to return a single sign

The initial screen shows the divinators and the single sign

There is a button in the lower right that says Personalize which takes you to a form for personalizing input to horoscope, tarot and iching. After personalization, a new main screen w/ new readings appears
