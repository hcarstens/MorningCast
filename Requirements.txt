# MorningCast Requirements

## Prerequisites

- **Node.js**: v18+ (tested with v25.1.0)
- **npm**: v8+ (tested with v11.6.2)
- **macOS/Linux/Windows**: Cross-platform compatible

## Quick Start

The app is already set up in the `morning-cast-app/` directory. To run:

```bash
cd morning-cast-app
npm run dev
```

Then visit `http://localhost:3000`

## Dependencies

### Core Framework
- **Next.js 15**: React framework with App Router
- **React 19**: UI library
- **TypeScript**: Type safety
- **Tailwind CSS**: Utility-first styling

### UI Components
- **shadcn/ui**: Pre-built component library
  - button, card, input, label, sheet, select, textarea, badge
- **Lucide React**: Icon library
- **Framer Motion**: Smooth animations

### Development Tools
- **ESLint**: Code linting
- **PostCSS**: CSS processing
- **Autoprefixer**: CSS vendor prefixes

## Installation from Scratch

If you need to recreate the environment:

```bash
# Install Node.js (macOS)
brew install node

# Create Next.js project
npx create-next-app@latest morning-cast-app --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
cd morning-cast-app

# Initialize shadcn/ui
npx shadcn@latest init -d

# Add UI components
npx shadcn@latest add button card input label sheet select textarea badge

# Install additional dependencies
npm install framer-motion lucide-react

# Start development server
npm run dev
```

## Features

- **Daily Divination with Olivia**: Combines Horoscope, Tarot, and I Ching readings
- **Personalization**: Name, birthdate, favorite number, intention, and theme
- **Deterministic Randomness**: Consistent daily readings using seeded PRNG
- **Responsive Design**: Works on desktop and mobile
- **Local Storage**: Preferences saved in browser
- **Three Themes**: Serene, Mystic, and Bold color schemes

## Environment Configuration

The project includes comprehensive environment configuration:

- **`.env`**: Default application settings and feature flags
- **`.env.local.example`**: Template for local development overrides
- **`morning-cast-app/.env`**: Next.js-specific configuration

### Customization
Copy `.env.local.example` to `.env.local` and modify as needed:

```bash
cp .env.local.example .env.local
# Edit .env.local with your preferences
```

Key settings:
- `NEXT_PUBLIC_DEFAULT_THEME`: Default visual theme (Serene/Mystic/Bold)
- `NEXT_PUBLIC_SHOW_SEED_DEBUG`: Show/hide debug information
- `NEXT_PUBLIC_ENABLE_ANIMATIONS`: Enable/disable UI animations
- `PORT`: Development server port (default: 3000)

## Troubleshooting

- **Port 3000 in use**: Change port with `npm run dev -- -p 3001` or set `PORT=3001` in `.env.local`
- **Build errors**: Ensure Node.js v18+ and all dependencies installed
- **UI components missing**: Re-run `npx shadcn@latest add [component-name]`
- **Styles not loading**: Verify Tailwind CSS is properly configured
- **Environment issues**: Check `.env` files are properly formatted (no spaces around =)
